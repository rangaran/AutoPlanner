♡ this is how we created our project ♡

Starting with a Python code, we were able to create a program that could parse through PDF files and extract specific dates noted in the file. We then converted the dates into a .csv file, which can be uploaded to Google Calendar. We created a browser extension, which allowed ease of use for consumers to easily and automatically input class schedules and important dates into their personal calendar.
